# WP-Composer-Homstead

Develop WordPress Sites with Composer and Homestead.

#Installation
`git clone git@github.com:gpspake/WP-Composer-Homstead.git`  
`cd WP-Composer-Homstead`  
`composer install`  
`vagrant up`
